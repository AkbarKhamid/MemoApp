# Memo app - submission for WashWat coding assessment

## ▶️ Usage

1. [Setting up IOS and Android development environment](https://reactnative.dev/docs/environment-setup)

2. Start working

```bash
## install deps
yarn

## install ios pods
yarn setup:ios

## start metro
yarn start

## ios
yarn ios

## android
yarn android

```
